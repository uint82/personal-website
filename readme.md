# Personal Website

This repository contains the source code for my personal website. It is a static site built to be lightweight and performant.

## Tech Stack

- **Build Tool:** Vite
- **Language:** TypeScript
- **Markup/Styling:** HTML, CSS

## Prerequisites

- Node.js (LTS version recommended)
- npm or yarn

## Installation

1.  Clone the repository:

    ```bash
    git clone [https://github.com/uint82/personal-website.git](https://github.com/uint82/personal-website.git)
    cd personal-website
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

## Development

To start the local development server:

```bash
npm run dev
```
